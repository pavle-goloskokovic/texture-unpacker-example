# 🗃️💁 TextureUnpacker Example

TextureUnpacker is a Node.js tool written in TypeScript which can be used to
unpack `.png` sprite sheets packed with [TexturePacker](https://www.codeandweb.com/texturepacker/) into separate sprites if provided with corresponding `.json` or `.plist` data file.

This repository serves as a **practical working example** for using the (soon-to-be-published) `texture-unpacker` npm package. You can install the package in your own project and refer to the files here for guidance on programmatic and command-line usage.
